# 北九州四天三夜旅遊規劃

## 行程概要

| 日期 | 路線 | 主題 |
|---|---|---|
| 9/3（四）| 北九州機場 → 太宰府 → 柳川 → 福岡 | 歷史寺社・水鄉・鰻魚飯 |
| 9/4（五）| 博多・天神 → 小倉 → 皿倉山 | 逛街美食・北九州夜景 |
| 9/5（六）| 小倉 → 門司港 → 唐戶市場 | 海鮮市場・港町復古・燒咖哩 |
| 9/6（日）| 小倉市區 → 北九州機場 | 小倉城・旦過市場・返程 |
